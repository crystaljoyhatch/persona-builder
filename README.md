## Why I Built This

I run a four-person team that operates at the output of fifteen. Part of how
that works is that everyone who works with Claude has a personal OS loaded
into their projects. Claude knows each person. It does not need to be
re-introduced.

I built this for my team first. Then my sister used it. Then a few colleagues.
Then I realized the setup process itself was the thing worth sharing.

The three-file structure, the interview flow, the personality layer, the proxy
mode for building teammate personas -- none of that existed in one place before.
Now it does.
